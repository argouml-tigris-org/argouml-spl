Description

ArgoUml-SPL is a project which aims to create a product line software for the ArgoUML. With this Software Product Line shall be possible drive customizable versions of this application, thus allowing their users can obtain customized versions of ArgoUML for their needs.

Building the application by their own way, the users may obtain a version of ArgoUML that will contain only what really wants, without the necessity to obtain a package with various functionalities which will not be used, which leads the economy of space for installation, less traffic to the distribution of installation and smaller consumption of memory and processing obtained with no need of loading subsystems which would not be used.

Mission

The main objective of ArgoUML-SPL is the creation of a Software Product Line for the ArgoUML.

The scope of the this project is bounded in extract the main features of ArgoUML in order to provide the greatest unique versions for this application.

How can I collaborate?

Everybody is welcome to help with developing and testing ArgoUML-SPL. All suggestions can be send through the Developers Mailing List.

We used ArgoUML v0.28.1 to create the initial version of ArgoUML, so some fixed bugs in this version still exists in ArgoUML-SPL. Beyond extract features, we are looking for people who wants to help us with the merges between the last stable version of ArgoUML and the last version of ArgoUML-SPL.
